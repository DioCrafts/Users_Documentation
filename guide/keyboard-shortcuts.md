# Keyboard shortcuts

A few keys make AXIOM much faster. Press <kbd>?</kbd> anywhere to see this list inside
the app.

## Everywhere

| Keys | Action |
|---|---|
| <kbd>⌘K</kbd> / <kbd>Ctrl-K</kbd> | Open search & the command palette |
| <kbd>⌘J</kbd> / <kbd>Ctrl-J</kbd> | Open the Copilot |
| <kbd>?</kbd> | Show keyboard shortcuts |
| <kbd>Esc</kbd> | Close the current overlay or panel |

## Quick actions

These work from the command palette and as quick keys:

| Keys | Action |
|---|---|
| <kbd>G</kbd> then <kbd>C</kbd> | New project |
| <kbd>E</kbd> | Open Reports |
| <kbd>T</kbd> | Toggle light / dark theme |

## On the graph (and other canvases)

| Input | Action |
|---|---|
| **Drag** the canvas | Pan |
| **Scroll** | Zoom in / out |
| **Drag** a node | Move it |
| **Click** a node | Select it |
| **Double-click** a node | Expand its connections, or open its 360° profile |
| **Arrow keys** | Move between nodes |
| <kbd>Enter</kbd> / <kbd>Space</kbd> | Activate the focused node |
| <kbd>Esc</kbd> | Clear the selection |

The same arrow-key navigation works on the [map](/guide/geospatial) and the linked
analysis panels: arrows move between items, <kbd>Enter</kbd> selects, <kbd>Esc</kbd>
clears.

## Editors

In the [Notepad](/guide/reports#notepad) and rich-text report blocks, press
<kbd>/</kbd> to open the block menu and insert headings, lists, callouts, tables,
images, and more without leaving the keyboard.
